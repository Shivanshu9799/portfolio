## Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
}
```

### Update Content
Edit sections in `index.html`:
- Hero section
- About section
- Research projects
- Skills
- Achievements
- Contact information
